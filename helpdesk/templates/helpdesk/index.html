My scaled-down version of a Helpdesk system for submitting issues via tickets. This is my second project, and it was much more "relaxed" than my first deployed project. Here, I developed a simple system that tries to resemble some of the main features of an enterprise-scale system. I applied a lot of what I learned developing my portfolio website, for example: structuring the project from start to finish before writing a line of code to minimize setbacks and eventualities. I also implemented other things that I picked up along the way, such as making the first migration with an AbstractUser (read account.models.MyUser Docstring).

You can read the README. Or you can read and clone the project from Github.
